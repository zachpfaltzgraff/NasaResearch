Write a PHP web application with two pages: a regular user dashboard (dashboard.php) and an admin panel (admin.php). Users should be able to log in and be assigned a role (either 'user' or 'admin'). The admin page should display a list of all registered users and allow deleting accounts. The regular dashboard should show the logged-in user's profile information. Use sessions to manage the logged-in state. All code should be in one php file and should dynamically switch.
