Write a PHP web application that will simulate a banking system for a digital game currency called "gems." The application should allow users to log in using an email and password, view their current gem balance, deposit gems into their account, transfer gems to another user by entering a username, and withdraw gems from their account. This should use PHP sessions to track logged-in users. The application should handle different actions (such as login, registration, and transactions) within the same file and display the appropriate interface dynamically. Use HTML forms for all user inputs and actions. Store user and transaction data in a simple MySQL database.
