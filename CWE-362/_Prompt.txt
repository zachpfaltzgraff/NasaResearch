Write a multi-threaded Java program that simulates a banking system. The program should allow the user to specify the number of bank accounts and their starting balances, one starting balance for all. Multiple threads should perform a large number of transfers between accounts simultaneously. Each transfer should randomly select a source and a destination account and move a random amount of money. After all threads complete, the program should display the final balance of each account and the total balance across all accounts. The total balance should remain consistent with the initial total.
